# 捐赠功能说明

## 如何添加捐赠二维码

1. 准备您的微信或支付宝收款二维码图片
2. 将图片保存为 `donate_qr.png`
3. 将图片放置在项目根目录下（与 `gui.py` 同级目录）
4. 重新启动程序，点击"💝 捐赠支持"按钮即可看到二维码

## 图片要求

- 文件名：`donate_qr.png`
- 格式：PNG（推荐）或 JPG
- 大小：建议 300x300 像素或更大（程序会自动缩放）
- 位置：项目根目录

## 功能说明

点击配置区域的"💝 捐赠支持"按钮后，会弹出捐赠对话框，显示：
- 项目信息
- 捐赠感言
- 二维码图片（如果已添加）
- 使用说明

如果未找到二维码图片，对话框会显示提示信息，指导您如何添加。
